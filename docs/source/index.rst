==============================================================
Xena OpenAutomation Test Suite Application User Manual
==============================================================

Xena OpenAutomation Test Suite Application is a web-based automated test methodology suite application including all Xena's test suites for test specialists to perform benchmarking tests, service validation, and more.

Executive Briefing
==================

Powered by `XOA Python API <https://docs.xenanetworks.com/projects/xoa-python-api>`_, `XOA Core <https://docs.xenanetworks.com/projects/xoa-core>`_, `XOA Test Suite Plugins <https://github.com/xenanetworks/open-automation-test-suites>`_, and `XOA Converter <https://docs.xenanetworks.com/projects/xoa-config-converter>`_, Xena OpenAutomation (XOA) Test Suite Application includes all Xena's automated test suites, which overlays various Xena hardware and virtual :term:`TGA`, for test specialist to verify design, quality, performance, and more with ease of integration into a test automation framework.

XOA Test Suite Application provides standardized test methodologies for performance benchmarking, service validation, conformance, and is continuously adding new test methodologies.

XOA Test Suite Application persistent data storage and is deployed on-premises to keeping all data within the organization's network boundaries.

This guide provides installation instructions and user guide to help get your XOA Cluster up and running as quickly as possible and walk you through how to use the GUI application is various ways.


Available Test Suites
=====================

.. important::

    In version 1.0, only :term:`XOA2544` is available.

    :term:`XOA2889`, :term:`XOA3918`, and :term:`XOA1564` will be available in future releases.
